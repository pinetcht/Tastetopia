### Installation
To run our app follow these steps

1. Clone the repo
```bash
git clone {repo url}
```
2. Run the frontend code
```bash
cd frontend
npm install
npm install firebase
npm run dev
```
2. Run the backend code
```bash
cd backend
npm install
npm install dotenv cors express cookie-parser request firebase
npm start
```
